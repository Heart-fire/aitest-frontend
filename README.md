### 项目介绍
项目深度融合业务场景，采用 Vue 3、Spring Boot、Redis、ChatGLM AI、RxJava 和 SSE 等技术栈。用户可以通过 AI 快速创建并发布多种答题应用，支持应用检索、在线答题，以及基于评分算法或 AI 生成回答总结。平台还为管理员提供了应用审核、全站内容集中管理及数据统计分析功能。
